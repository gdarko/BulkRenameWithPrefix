# BulkRenameWithPrefix

Simple C program that renames bulk files with given prefix

To start using it first, you need to compile with GCC/MinGW and then open it with Terminal or CMD

```c
bulkrenamer.exe C:\\MYDIR brp _
```

### Instructions:
"C:\\MYDIR" is the location of your files

"brp" is the prefix for the files. For example file.php will be renamed to file_brp.php

"_" Is the separator that will be used. For example file.php use _ to add the prefix
